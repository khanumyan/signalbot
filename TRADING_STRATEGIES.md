# Торговые стратегии для криптовалютных сигналов

## Обзор проекта

Данный проект представляет собой автоматизированную систему анализа криптовалютных рынков с использованием шести различных торговых стратегий. Система работает в режиме реального времени, анализируя более 400 криптовалютных пар и отправляя торговые сигналы через Telegram.

## Торговые стратегии

### 1. MTF (Multi-TimeFrame) Strategy - Основная стратегия
**Команда:** `crypto:analyze`  
**Частота выполнения:** Каждые 12 минут  
**Таймфреймы:** 5m, 15m, 1h  
**Тип стратегии:** Мультитаймфреймовая с фильтрацией тренда

**Описание:**
- **Основной таймфрейм:** 15 минут - для генерации базовых сигналов
- **Старший таймфрейм (HTF):** 1 час - для определения общего тренда
- **Младший таймфрейм (LTF):** 5 минут - для подтверждения сигналов

**Индикаторы:**
- RSI (14 периодов) на всех таймфреймах
- EMA (50 периодов) для фильтрации тренда
- Bollinger Bands (20 периодов, 2 стандартных отклонения)
- ATR (14 периодов) для расчета волатильности
- Объемный анализ

**Логика сигналов:**
- Сигналы генерируются на основе RSI экстремумов (≤20 для покупки, ≥80 для продажи)
- HTF тренд должен поддерживать направление сигнала
- LTF подтверждает сигнал на младшем таймфрейме
- Сила сигнала зависит от совместимости с трендом и подтверждения

### 2. EMA + RSI + MACD Strategy - Универсальная трендовая стратегия
**Команда:** `crypto:ema-rsi-macd`  
**Частота выполнения:** Каждые 15 минут  
**Таймфрейм:** 15 минут  
**Тип стратегии:** Трендовая с подтверждением импульса

**Описание:**
- Использует пересечения EMA для определения тренда
- MACD подтверждает импульс движения
- RSI фильтрует перекупленность/перепроданность

**Индикаторы:**
- EMA 20 и EMA 50 для определения тренда
- MACD (12, 26, 9) для подтверждения импульса
- RSI (14 периодов) для фильтрации

**Логика сигналов:**
- **Покупка:** Цена выше EMA20, EMA20 > EMA50, MACD > 0, RSI < 70
- **Продажа:** Цена ниже EMA20, EMA20 < EMA50, MACD < 0, RSI > 30
- Сила сигнала зависит от RSI и силы MACD гистограммы

### 3. Bollinger Bands + RSI Strategy - Контртрендовая стратегия
**Команда:** `crypto:bollinger-rsi`  
**Частота выполнения:** Каждые 20 минут  
**Таймфрейм:** 15 минут  
**Тип стратегии:** Контртрендовая с отскоками от границ

**Описание:**
- Использует отскоки от границ Bollinger Bands
- RSI подтверждает перекупленность/перепроданность
- Подходит для боковых рынков

**Индикаторы:**
- Bollinger Bands (20 периодов, 2 стандартных отклонения)
- RSI (14 периодов)
- ATR для расчета стоп-лоссов

**Логика сигналов:**
- **Покупка:** Цена касается нижней границы + RSI < 30
- **Продажа:** Цена касается верхней границы + RSI > 70
- Сила сигнала зависит от экстремальности RSI

### 4. EMA + Stochastic Strategy - Скальпинговая стратегия
**Команда:** `crypto:ema-stochastic`  
**Частота выполнения:** Каждые 10 минут  
**Таймфрейм:** 5 минут  
**Тип стратегии:** Скальпинг с импульсными сигналами

**Описание:**
- Быстрые сигналы для краткосрочной торговли
- Использует пересечения EMA и выход Stochastic из зон перекупленности/перепроданности
- Оптимизирована для внутридневной торговли

**Индикаторы:**
- EMA 9 и EMA 21 для быстрого тренда
- Stochastic (14, 3, 3) для определения импульса
- ATR для узких стоп-лоссов

**Логика сигналов:**
- **Покупка:** EMA9 > EMA21 + Stochastic выходит из перепроданности (K > 20)
- **Продажа:** EMA9 < EMA21 + Stochastic выходит из перекупленности (K < 80)
- Сила сигнала зависит от разности между K и D линиями

### 5. SuperTrend + VWAP Strategy - Внутридневная трендовая стратегия
**Команда:** `crypto:supertrend-vwap`  
**Частота выполнения:** Каждые 30 минут  
**Таймфрейм:** 15 минут  
**Тип стратегии:** Внутридневная трендовая с возвратом к справедливой цене

**Описание:**
- SuperTrend определяет направление тренда
- VWAP показывает справедливую цену
- Сигналы генерируются при возврате цены к VWAP

**Индикаторы:**
- SuperTrend (10 периодов, множитель 3)
- VWAP (Volume Weighted Average Price)
- ATR для расчета риска

**Логика сигналов:**
- **Покупка:** SuperTrend показывает восходящий тренд + цена возвращается к VWAP сверху
- **Продажа:** SuperTrend показывает нисходящий тренд + цена возвращается к VWAP снизу
- Сила сигнала зависит от расстояния от VWAP

### 6. Ichimoku + RSI Strategy - Долгосрочная трендовая стратегия
**Команда:** `crypto:ichimoku-rsi`  
**Частота выполнения:** Каждый час  
**Таймфрейм:** 1 час  
**Тип стратегии:** Долгосрочная трендовая с облачной поддержкой

**Описание:**
- Использует облако Ишимоку для определения тренда
- RSI подтверждает сигналы
- Подходит для позиционной торговли

**Индикаторы:**
- Tenkan-sen (9 периодов)
- Kijun-sen (26 периодов)
- Senkou Span A и B (52 периода)
- RSI (14 периодов)

**Логика сигналов:**
- **Покупка:** Цена выше облака + Tenkan > Kijun + RSI поддерживает (40-70)
- **Продажа:** Цена ниже облака + Tenkan < Kijun + RSI поддерживает (30-60)
- Сила сигнала зависит от RSI и расстояния от облака

## Анализируемые активы

Система анализирует более 400 криптовалютных пар, включая:
- Основные криптовалюты: BTC, ETH, BNB, ADA, SOL, DOT, AVAX
- DeFi токены: UNI, AAVE, COMP, SUSHI, CRV
- Мем-коины: DOGE, SHIB, PEPE, FLOKI
- Новые токены: TON, WIF, BONK, WLD

## Управление рисками

### Стоп-лоссы и тейк-профиты
- **Скальпинг (EMA+Stochastic):** RR 1:1 до 1:2
- **Краткосрочная торговля (MTF, EMA+RSI+MACD):** RR 1:1.5 до 1:3
- **Среднесрочная торговля (Bollinger+RSI, SuperTrend+VWAP):** RR 1:2 до 1:3
- **Долгосрочная торговля (Ichimoku+RSI):** RR 1:1.5 до 1:3

### Фильтрация сигналов
- Предотвращение дублирования сигналов в течение 15 минут
- Проверка изменения силы сигнала перед отправкой
- Учет общего тренда на старших таймфреймах
- Анализ объемов для подтверждения сигналов

## Техническая реализация

### Расписание выполнения
- **Скальпинг:** Каждые 10 минут (5m таймфрейм)
- **Краткосрочная торговля:** Каждые 12-15 минут (15m таймфрейм)
- **Среднесрочная торговля:** Каждые 20-30 минут (15m таймфрейм)
- **Долгосрочная торговля:** Каждый час (1h таймфрейм)

### Мониторинг и логирование
- Все сигналы сохраняются в базе данных
- Логирование всех операций в файлы
- Отправка уведомлений об отсутствии сигналов
- Тестирование Telegram соединения перед отправкой

### Надежность системы
- Защита от перекрывающихся выполнений команд
- Обработка ошибок API
- Автоматическое восстановление после сбоев
- Фоновое выполнение для предотвращения блокировок

## Заключение

Система представляет собой комплексное решение для автоматизированной торговли криптовалютами, объединяющее различные подходы к анализу рынка. Мультистратегический подход позволяет адаптироваться к различным рыночным условиям, от скальпинга до долгосрочной позиционной торговли.
